## ⚙️ Como Executar

### Pré-requisitos

- GCC instalado
- Sistema operacional Windows, Linux ou macOS

### Clonar o repositório

```bash
git clone https://github.com/matheusfideles-stack/Tech-Nexus.git
```

### Compilar o projeto

```bash
gcc main.c -o tech-nexus
```

### Executar

#### Windows

```bash
tech-nexus.exe
```

#### Linux/macOS

```bash
./tech-nexus
```
